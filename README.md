# 🎓 Dự Án Quản Lý Sinh Viên: Cơ sở dữ liệu SQL + Lập Trình Trực Quan C#

Dự án hoàn thiện môn Cơ sở dữ liệu & Lập trình trực quan. 

## 🚀 Tính Năng Chính
* Quản lý thông tin Khoa, Lớp và dữ liệu Sinh viên (Thêm, Sửa, Xóa, Hiển Thị).
* Liên kết đồng bộ 3 bảng CSDL dưới SQL Server.
* **Tính năng nâng cao:** Tìm kiếm thông tin sinh viên tự động qua kết nối ngoại vi cổng **COM (Serial Port)** từ Terminal.

## 🛠️ Công Nghệ Sử Dụng
* **Ngôn ngữ:** C# WinForms (.NET Framework)
* **Cơ sở dữ liệu:** Microsoft SQL Server
* **Công cụ phát triển:** Visual Studio 2022

## 📂 Cấu Trúc Kho Lưu Trữ
* `/Sql Sever`: Chứa file script `.sql` khởi tạo database và nạp dữ liệu mẫu.
* `/Visual Studio 2022`: Chứa toàn bộ mã nguồn giao diện, file handle code C#.
* `QuanLySinhVien.sln`: File khởi chạy toàn bộ giải pháp hiển thị trực  trên Visual Studio.
